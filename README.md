# Quiz-Game
Quiz Game is an Android application designed to provide a fun and interactive quiz experience. The app allows users to authenticate securely using Firebase Authentication and participate in multiple-choice quizzes while tracking their performance.

Features: User Registration and Login, Firebase Email/Password Authentication, Google Sign-In Integration, Multiple-Choice Quiz Questions, Real-Time Score Tracking with Firebase Realtime Database, Result Screen Showing Correct and Wrong Answers, Attractive UI with Custom Graphics and Animations, Responsive Layout for Different Android Devices


Technologies Used: Java, Android Studio, Firebase Authentication, Firebase Realtime Database, XML Layouts, Google Sign-In API, Future Improvements, Difficulty Levels, Leaderboards, Timer-Based Questions, Category Selection, Sound Effects and Animations, Question Randomization
